# Ideas Lab — Frontend

Aplicação frontend construída com Vite, React e TypeScript, estilizada com Tailwind CSS

## Tecnologias
- React 19+
- Vite 7
- TypeScript 5
- Tailwind CSS 4
- ESLint

## Requisitos
- Node.js 18+ (recomendado 20+)
- npm 9+

## Como começar
1. Instale as dependências
   ```bash
   npm install
   ```
2. Ambiente de desenvolvimento
   ```bash
   npm run dev
   ```
   Acesse a URL exibida pelo Vite (ex.: http://localhost:5173).
3. Build de produção
   ```bash
   npm run build
   ```
4. Preview do build
   ```bash
   npm run preview
   ```

## Scripts (package.json)
- `npm run dev` — inicia o servidor de desenvolvimento do Vite
- `npm run build` — compila TypeScript e gera o build de produção
- `npm run preview` — serve o build localmente para inspeção
- `npm run lint` — executa ESLint
